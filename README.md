# 🚀 iNOTHING

> **Beautifully useless.** A premium Android app that does absolutely nothing — in the most stunning way possible.

---

## ✨ Features

| Module | Status | Description |
|--------|--------|-------------|
| 🔢 **iCOUNTING** | ✅ Live | Count through 1000 pages with golden milestones every 100 pages |
| 🫁 **iBREATHING** | 🔜 Coming Soon | Guided breathing exercises |
| 👆 **iTAPPING** | 🔜 Coming Soon | Speed tapping challenge |
| ⏳ **iWAITING** | 🔜 Coming Soon | The art of doing nothing |

## 🎨 Design

- **Apple glassmorphism** — frosted glass cards, semi-transparent layers
- **Animated gradient backgrounds** with floating color orbs
- **Smooth page transitions** with slide + fade effects
- **Golden milestone celebrations** with sparkle particle animations
- **Dark, immersive UI** — edge-to-edge, no action bar

## 📱 Tech Stack

- **Kotlin** + **Jetpack Compose**
- **Material 3** Design System
- **Single Activity** Architecture
- **minSdk 24** — supports 95%+ of Android devices

## 🔒 Privacy

iNOTHING collects **zero data**. No analytics, no tracking, no permissions.
